# tree-sitter-atob

[A=B](https://esolangs.org/wiki/A%3DB) grammar for
[tree-sitter](https://tree-sitter.github.io/tree-sitter).

A=B is a string-rewriting esolang from [game of the same
name](https://store.steampowered.com/app/1720850/AB/).

## Status

This toy grammar exists solely for practice purpose for the author. It should
be considered incomplete, potentially incorrect, insufficiently tested, and
almost certainly a poor life choice. It is not supposed to be used anywhere,
especially in any environment where "works on my machine" is considered an
inadequate safety case.
